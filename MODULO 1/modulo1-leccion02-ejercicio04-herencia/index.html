Lenguajes de Programación

¿Qué es un lenguaje de programación?

Un lenguaje de programación es un lenguaje formal (o artificial, es decir, un lenguaje con reglas gramaticales bien definidas) que le proporciona a una persona la capacidad de escribir (o programar) una serie de instrucciones o secuencias de órdenes en forma de algoritmos con el fin de controlar el comportamiento físico o lógico de una computadora, de manera que se puedan obtener diversas clases de datos o ejecutar determinadas tareas. A todo este conjunto de órdenes escritas mediante un lenguaje de programación se le denomina programa.

Los lenguajes de programación están formados por un conjunto de símbolos, reglas gramaticales y semánticas, que en conjunto definen las estructuras válidas del lenguaje y su significado.

Los distintos lenguajes de programación pueden clasificarse atendiendo a distintos criterios; clasificación histórica, lenguaje de alto y bajo nivel, clasificación por paradigmas y clasificación por propósito. Para más información: Lenguaje de programación - Wikipedia

Lenguajes de programación Frontend

La programación Front-end se refiere a la parte del sitio web encargada de la interacción con el usuario. Para ello, el programador front-end se ayuda de una serie de lenguajes de programación; HTML, CSS y JavaScript:

En primer lugar tenemos HTML o HyperText Markup Language, cuyo funcionamiento se basa en la utilización de etiquetas que estructuran y organizan el contenido de la web.

Asociado a HTML encontramos CSS, que se encarga del formato y diseño visual de las páginas web escritas en html.

Por último, Javascript que se encarga de hacer dinámico el diseño de la web. Utiliza animaciones y acciones interactivas con el usuario. También interactúa con los lenguajes HTML y CSS.

Lenguajes de programación Backend

La programación Back-end se refiere a la parte de un sistema software con el que el usuario no interactua. Un programador back-end utiliza una gran variedad de lenguajes de programación. A continuación se muestra un listado con algunos ejemplos

  • PHP
  • Python
  • Java
  • C++
